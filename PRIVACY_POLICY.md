## 3. How We Use Your Information

Since we do not collect any personal information, there is no data usage to report. All AI processing happens **entirely on your local device** using offline AI models.

### 3.1 File Processing
When you use the document reading feature:
- Files are processed **locally** on your device
- No file contents are uploaded to external servers
- Extracted text is used only for your AI conversation
- Temporary data is deleted after processing

### 3.2 Web Search Feature (Optional)
If you enable the optional web search enrichment feature:
- Search queries are sent to a search API service
- **We do not control or log these search queries**
- The search service may have its own privacy policy
- You can disable this feature at any time in settings

### 3.3 Voice Commands (Optional)
If you use voice commands:
- Voice processing happens **locally** on your device
- No audio is recorded or transmitted
- Voice data is converted to text locally and then discarded
